# 통합 학습 관리 시스템

학생과 교사를 위한 통합된 학습 관리 시스템입니다.

## 기능

### 학생 기능
- 퀘스트 수행 및 보상 획득
- 가챠를 통한 캐릭터 수집
- 친구들과의 배틀 및 경쟁
- 개인 진도 및 성취도 확인
- 컬렉션 관리

### 교사 기능
- 클래스 및 학생 관리
- 퀘스트 생성 및 승인
- 레이드 생성 및 관리
- 학생 진도 모니터링
- 성적 및 활동 분석

## 기술 스택

- React 18
- TypeScript
- Vite
- Tailwind CSS
- Radix UI

## 설치 및 실행

```bash
# 의존성 설치
npm install

# 개발 서버 실행
npm run dev

# 프로덕션 빌드
npm run build
```

## 프로젝트 구조

```
src/
├── components/
│   ├── student/          # 학생용 컴포넌트
│   ├── teacher/          # 교사용 컴포넌트
│   └── ui/               # 공통 UI 컴포넌트
├── utils/                # 유틸리티 함수
└── styles/               # 스타일 파일
```

## 역할별 접근

1. **학생**: 역할 선택 → 학생 로그인 → 학생 대시보드
2. **교사**: 역할 선택 → 교사 로그인 → 교사 대시보드
